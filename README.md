## 프로젝트 내용

사용자는 게임을 통해 얻은 포인트를 확인하고, 포인트를 원화 또는 코인으로 출금한다. 그동안의 출금 내역과 포인트 내역을 확인하고 초대내역을 확인한다.

관리자는 사용자가 신청한 출금을 수락 또는 거절하고 이용중인 모든 사용자의 정보를 확인한다.
